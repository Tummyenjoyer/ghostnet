# GhostNet Fishing

## Technologien & Versionen

- Java: 21.0.11 (Temurin)
- Spring Boot: 3.5.14
- Spring Data JPA + Hibernate (automatisch via Spring Boot)
- Maven: 3.9.16
- MySQL: 8.0.41
- Thymeleaf (automatisch via Spring Boot)

## Datenbank Setup

1. MySQL installieren
2. Datenbank erstellen: CREATE DATABASE ghostnet;
3. database_dump.sql importieren
4. application.properties anpassen (Passwort eintragen)

## App starten

1. Projekt in IntelliJ öffnen
2. GhostnetApplication.java ausführen
3. Browser öffnen: localhost:8080
